前言

Markdown是一种轻量级标记语言，于2004年由创始人John Gruber发布，采用并不复杂的语法，可以书写简洁美观带有格式的文本。

基本语法

关于符号
重要的事情提前说：Markdown中的所有标记符号都是英氏符号，也就是输入法为输入英文状态下的时候。(尤其警惕星号*，竖向分隔线|，感叹号!，括号()，等等…中英文的这几个有点像)

段落与换行

  1. 空行可以用来段落分割，输入回车Enter换行即可；
  2. 段落内换行可以使用<br>来换行，通常可以用在markdown表格内换行;
  3. 某些markdown在写完上一行文本后输入Enter换行依然未能换行生效,在上一行末尾加2个连续空格即可。

标题层级

Markdown使用符号 # 来表示标题的层级，几个 # 号就是几级标题。

# 一级标题
## 二级标题
### 三级标题
#### 四级标题

字体样式

  1. 字体加粗：加粗的内容使用前后2个连续的星号* 包围起来,例如：
    输入内容 渲染效果
    **这是加粗的内容** 这是加粗的内容

注意：如果加粗的内容有*,#,-,这一类markdown语法有关的字符，可以用英式反斜杠\加在前面转义;某些Markdown编译器对*号解析有点问题，如果加粗效果微生效，外侧的2个*号前后加空格即可,和未加粗的内容分隔开。

  1. 字体斜体：斜体的内容使用前后1个星号*包围或者前后1个下划线_包围起来,例如：

    输入内容 渲染效果
    *这是斜体的内容* 这是斜体的内容
    _这也是斜体的内容_ 这也是斜体的内容

  2. 字体高亮：高亮的内容使用前后1个英式反引号包围起来(反引号`在键盘左上角,数字1左边)，例如:

    输入内容 渲染效果
    `这是高亮的内容` 这是高亮的内容

  3. 字体删除线：对删除线的内容前后2个连续的英式波浪号 ~包围起来，例如：

    输入内容 渲染效果
    ~~这是删除线的内容~~ 这是删除线的内容

  4. 字体的组合:以上的效果可以组合起来使用：

    输入内容 渲染效果
    ***加粗和斜体*** 加粗和斜体
    ~~**删除线和加粗~~** 删除线和加粗
    ~~*删除线和斜体*~~ 删除线和斜体
    ~~***加粗, 斜体和删除线***~~ 加粗, 斜体和删除线
    **`加粗和高亮`** 加粗和高亮

列表样式

列表分为无序列表和有序列表：无序列表用英式符号短横-、加号+、或星号* 后接空格来表示；有序列表使用数字加点接空格(例如"1. “)来表示。
输入内容：

1
2
3
4
5
6
7
8

- 短横无序列表
- 短横无序列表
+ 加号无序列表
+ 加号无序列表
* 星号无序列表
* 星号无序列表
1. 数字有序列表
2. 数字有序列表

渲染效果：

  • 短横无序列表
  • 短横无序列表
  • 加号无序列表
  • 加号无序列表
  • 星号无序列表
  • 星号无序列表
  1. 数字有序列表
  2. 数字有序列表
  3. 数字有序列表

提示：

  • 列表一般会自动缩进，而且写完一条列表内容后，按下Enter换行，markdown会自动新建下一个列表
  • 列表可以嵌套使用

代码段控制

代码段的高亮使用3个连续的反引号 ` 来标识代码段，并后接语言类型(针对语法高亮)，结尾处用3个反引号包围代码段。如果不接语言类型，也是可以的，不做任何语法的高亮。如下：

输入以下内容
```c++
#include
int main(){
std::cout « “hello world!\n”;
return 0;
}
```
